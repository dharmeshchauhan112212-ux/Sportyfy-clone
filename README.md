# 🎵 Spotify Clone

A modern and responsive **Spotify Web Player Clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of Spotify's web player with a clean user interface, responsive layout, and interactive design elements. It was developed as a front-end practice project to improve HTML and CSS skills.

---

## 🚀 Features

- 🎧 Spotify-inspired user interface
- 📱 Responsive layout
- 🏠 Sidebar navigation
- 📚 Music library section
- 🎵 Recently Played section
- 🔥 Trending music cards
- 📊 Featured Charts
- 🎼 Music player interface
- 🎨 Modern dark theme
- ⭐ Hover effects and smooth UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Font Awesome
- Google Fonts (Montserrat)

1. Clone this repository.

```bash

```

2. Open the project folder.

3. Double-click **index.html** or open it using **Live Server** in VS Code.
